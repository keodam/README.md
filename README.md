## 🚀 주요 기능 (Features)

- 💬 대면 커피챗 중계 및 매칭 시스템
- 🔍 커뮤니티 및 아티클 게시, 댓글/좋아요/링크 공유
- ⭐️ 리뷰 기반 신뢰도 시스템 (최신순 정렬 + 별점)
- 💰 추천인 & 미션 이벤트 기반 원두(포인트) 보상


## 👥 이런 분들께 추천해요 !

- 🤔 **진로·취업 고민이 있는 대학생**  
  → 선배 경험을 빠르게 듣고, 방향성을 찾고 싶은 분  

- 👫 **새로운 사람들과 네트워킹을 원하는 분**  
  → 편하게 대화하면서 관계를 넓히고 싶은 분  

- 📝 **실제 경험 기반의 정보를 찾는 분**  
  → 인터넷 검색이 아닌, 생생한 조언을 얻고 싶은 분  

- 🎯 **성장과 자기계발에 관심 있는 분**  
  → 다양한 대화 속에서 동기부여와 인사이트를 얻고 싶은 분
